# AGENTS.md

# Project Goals

This project is primarily a learning project.

Objectives:
- Learn FastAPI
- Learn React + TypeScript
- Learn PostgreSQL
- Learn Docker
- Learn CI/CD
- Learn Cloud deployment

# Planning
- Translate user requirements, rough ideas, and prompts into clear project documentation.
- Break down requirements into step-by-step implementation items.
- Identify unclear requirements, assumptions, dependencies, risks, and open questions.
- Propose a practical development sequence that supports incremental delivery.
- If there are features removed, check against the code to remind if there is any code needs to be removed.

# Documentation
- Update documentation when significant changes are made.
- Document assumptions.
- Keep design documents synchronized with implementation.
- Document API routes with brief description.
- Document database tables with schemas.

# Code Review
- Find the discrepency between design doc and actual code
- When reviewing the code that I wrote, note the discrepency between my code and the defined rules or structures.

# Testing
- Use pytest with FastAPI TestClient.
- Create tests for models, migrations, services, and API endpoints when relevant.
- Do not call external APIs in unit tests.
- Mock external API clients.
